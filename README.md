# oto_lkp

## 概述
android_lkp文件夹存放的是修改过的lkp的源代码，使得其可以运行在androidx86上面。

lkpebizzy是利用android_lkp目录制作，打包好的可以在androidx86上面运行的ebizzy测试用例。 

lkpnbench是利用android_lkp目录制作，打包好的可以在androidx86上面运行的nbench测试用例。
